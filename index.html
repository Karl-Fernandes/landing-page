This involved using complex ideas such as design patterns and planning ahead in order to predict what ideas I and my partner would need to be familiar with. On top of this, we also created a mrXAI which used Dijsktra's shortest algorithm in order to make the best move by considering the location of the closest detective.

Image Gallery

I designed this image gallery to collect and display cool pictures on a website. ToDo Application

In this project I created a To-Do App using React.js, JavaScript, HTML and CSS. I implemented functionalities such as adding, deleting and updating tasks as well as allowing these tasks to be checked using a tickbox. These functionalities persisted during changes, such as when the page was reloaded, by storing the data using localStorage so that these changes would not be lost when the page was refreshed.
